# Thesis-Supporting-Files
Python Codes, Streamlit application and files used on how AI can support Six Sigma process optimization.

# Detail Repository Contents

- `Bearing_Health_Monitoring.py`
  - The Python Code used for the Streamlit Simulation.

- `Complete_Python_Codes.ipynb`
  - Jupyter Notebook containing the complete Python Codes for the entire process of how AI can support Six Sigma process optimization.
  
  - Running Streamlit
   - Contains steps on how to run the Simulation.

- `bearing_analysis_dataset.csv`
  - Bearing dataset used for the bearing analysis.

- `customer_reviews.csv`
  - Customer reviews dataset used in the Define Phase.

- `requirements.txt`
  - Required Python libraries for the project
